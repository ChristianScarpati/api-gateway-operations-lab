# API Gateway Operations Lab

Demo project simulating an enterprise API platform with:
- API Gateway
- Auth service
- Customer service
- Payments service
- PostgreSQL
- Monitoring with Prometheus/Grafana
- Docker Compose setup

## Goal
Show practical knowledge related to:
- API Gateway operations
- service lifecycle
- REST/HTTP
- containers
- security
- monitoring
- troubleshooting
- technical documentation

## How to run
```bash
docker compose up -d
```
or 
docker compose up --build 

## How to stop
```bash
docker compose down
```
