# Project Scope

## Main goal
Build a small enterprise-style API platform demo aligned with DevOps, API operations, and service lifecycle practices.

## Components
- API Gateway
- auth-service
- customer-service
- payments-service
- PostgreSQL
- monitoring stack

## Focus areas
- REST / HTTP
- routing
- authentication
- observability
- troubleshooting
- operational documentation
- service lifecycle

## Out of scope
- Kubernetes
- OpenShift
- frontend
- advanced IAM
- event-driven architecture
